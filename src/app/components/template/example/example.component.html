Below these instructions is a Locale example template for NEURONE (in JSON format). All string values must be translated to the target language for NEURONE use (string keys must not be translated). Save the locale templates as locale-XX.json, where XX is a language code (like en or es-CL). Words between double brackets (like {{bookmarks}}) must not be translated, because they are variables that are replaced during NEURONE execution.

Below there is an example of a HTML Template for NEURONE. Modals are customizable pop-up content screens (inside the web browser) that can be used in some stages of the NEURONE study. These templates must be written in plain HTML and they can contain Twitter Bootstrap styling, NEURONE translation strings and/or AngularJS controller logic.

NEURONE translation strings must be called inside the template as {{'{'}}{{'{'}} "mystring" | translate {{'}'}}{{'}'}}, where mystring is a key inside the NEURONE Translation JSON file.

AngularJS controller logic from NEURONE can be used referencing the modal scope object. As a convention, all data passed to the modal must be loaded through the modal.fields object.

    <div class="container-fluid">
      <div class="row">
        <div class="col-lg-2 col-md-2 col-sm-3 col-xs-3 col-lg-offset-3 col-md-offset-3 col-sm-offset-2 col-xs-offset-2">
          <img src="images/avatars/{{'{'}}{{'{'}}modal.fields.avatar{{'}'}}{{'}'}}.png" class="img-responsive">
        </div>
        <div class="col-lg-4 col-md-4 col-sm-5 col-xs-5">
          <h2>Hey! I need your help!</h2>
    
          <p>We are making an article for our school newspaper and we need some help recovering information from our local network.</p>
    
          <p>Don't worry, in the following step I will give you some advice for making this task.</p>
    
          <p>Your help will be very appreciated.</p>
    
          <p>Sarah Jones<br>
          School Newspaper Chief Editor</p>
        </div>
      </div>
    </div>

Below there is an example of a HTML Template for NEURONE. HTML Templates are customizable content screens that can be used in some stages of the NEURONE study. These templates must be written in plain HTML and they can contain Twitter Bootstrap styling, NEURONE translation strings and/or AngularJS controller logic.

NEURONE translation strings must be called inside the template as {{'{'}}{{'{'}} "mystring" | translate {{'}'}}{{'}'}}, where mystring is a key inside the NEURONE Translation JSON file.

AngularJS controller logic from NEURONE can be used referencing the following scope objects:

  • tutorial on Tutorial Stage
  • instructions on Instruction Stage
  • affective on Affective Stage (SAM)
  • taskQuestions on Question Stage
